_ ^ _ (Power) (reporter)

This block will raise the first number to the power of the second number, which means it will multiply the first number by itself second number times.


Infinty (reporter)

This simply returns the value of Infinty.


-Infinty (reporter)

This simply returns the value of negative Infinty.


_ is finite? (boolean)

If the number is finite it will return true, and if the number is infinite it will return false.


Remainder of _ divided by _ (reporter)

It will return the remainder of a division problem where the first number is the dividend, and the second number is the divisor.


Cookie Extension Block

Set cookie with name:_ and value:_ (command)

It will set a cookie thats name is the first value (after "name:") and its value will be the second value (after"value")


Delete cookie with name:_ (command)

It will delete the cookie thats name is the input.


Value of cookie with name:_ (reporter)

This will return the value of the cookie thats name is the input.


delete all cookies (command)

This will delte all the cookies that were set with this extension.
